README
=======
ADS-powerflow是一个适用于主动配电系统的潮流计算程序。采用Distflow形式的潮流方程，便于进行网络重构计算。采用混合整数二阶锥或其他线性化方式加快潮流计算的速度，尤其适合作为配电网规划和日前调度运行的底层程序。
推荐MATLAB2016b和GAMS24.8以上版本
